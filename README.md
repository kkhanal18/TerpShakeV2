Full Stack social network application with portfolio and discussion board. Developed using **ReactJS+Redux** with **NodeJS (Express)**, **RESTful API**, **MongoDB**, and 
**JWT authentication**

[Live Demo](http://bit.ly/TerpShake)

## How to run this code
1. Make sure MongoDB is running on your system
2. Clone this repository
3. Update config/keys.js with your test values for JWT key 
4. Open command line in the cloned folder
* To install dependencies, run npm install
* To run the application for development, run npm run dev
5. Open localhost:5000 in the browser

## Frontend 
1. **JavaScript** (ES6)  
2. **ReactJS** with **React-Router**
3. **Redux.js** state management
3. **Bootstrap** framework 

## Backend

1. **Node.js** with **Express**
2. **REST API** with **MongoDB** database  
3. Login and authentication with **JWT**
